# Timetable Management System

A desktop/web application for creating, managing, and viewing academic timetables with conflict detection and schedule optimisation.

## Features

- Create and manage class schedules
- Faculty and room assignment
- Conflict detection for overlapping sessions
- Visual timetable grid view
- Export/print timetable

## Getting Started

```bash
git clone https://github.com/DrOneEyed/timetable-management-system.git
cd timetable-management-system
```

Open the application following the setup instructions in the `files/` directory.
